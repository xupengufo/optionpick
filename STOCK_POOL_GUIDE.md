# 📈 股票池自定义功能说明

## 🎯 功能概述

美股期权卖方推荐工具现在支持完全自定义的股票池配置，您可以通过多种方式选择要分析的股票：

## ⚡ 三种选择方式

### 1. 快速分类选择
- 📊 科技股：AAPL, MSFT, GOOGL, AMZN, TSLA, META, NVDA...
- 🏦 金融股：JPM, BAC, WFC, C, GS, MS, V, MA...
- 🏭 传统蓝筹：JNJ, PG, KO, PEP, WMT, HD, MCD, DIS...
- 💊 医疗生物：UNH, PFE, ABT, TMO, DHR, BMY, ABBV...
- ⚡ 能源股：XOM, CVX, COP, SLB, HAL, OXY...
- 📈 市场指数：SPY, QQQ, IWM, DIA, VTI...
- 📊 行业ETF：XLF, XLE, XLK, XLV, XLI...
- 🥇 大宗商品：GLD, SLV, USO, UNG, DBC...
- 📈 波动率：VIX, UVXY, SVXY, SQQQ, TQQQ...

### 2. 预设列表选择
- 从80+热门股票和ETF中多选
- 包含所有主要行业和板块
- 按字母顺序排列，方便查找

### 3. 自定义输入
- 直接输入任意美股代码
- 支持批量输入（每行一个）
- 自动验证代码有效性
- 显示有效/无效代码列表

## 🔧 使用方法

### 快速上手
1. **打开侧边栏"股票池"部分**
2. **选择分类**：从下拉菜单选择一个分类（如"🏛️ 科技股"）
3. **自动添加**：该分类下所有股票自动添加到选择列表
4. **开始分析**：点击"🚀 开始分析"按钮

### 精细控制
1. **预设选择**：从多选框中选择具体股票
2. **自定义输入**：在文本框中输入格式如：
   ```
   AAPL
   TSLA
   NVDA
   CUSTOM_STOCK
   ```
3. **验证代码**：点击"✅ 验证自定义代码"检查有效性
4. **查看选择**：在"当前选择的股票"中确认最终列表

### 管理选择
- **🗑️ 清空选择**：删除所有已选股票
- **🔄 重置默认**：恢复到系统默认的5只热门股票
- **实时显示**：显示当前选择的股票数量和代码

## 💡 使用技巧

### 1. 组合策略
```
科技股分析：选择"🏛️ 科技股"分类
市场对冲：同时选择SPY、QQQ、VIX
行业轮动：选择多个行业ETF进行对比
```

### 2. 自定义关注列表
```
在自定义输入框中输入：
AAPL    # 您关注的个股
TSLA    # 热门股票
SPY     # 市场基准
VIX     # 波动率对冲
```

### 3. 批量分析
- 一次最多可选择50只股票
- 建议单次分析5-15只股票以获得最佳体验
- 过多股票可能导致分析时间较长

## 🎯 实际应用场景

### 场景1：科技股期权策略
```
1. 选择"🏛️ 科技股"分类
2. 添加自定义股票如NVDA、AMD
3. 分析备兑看涨期权机会
4. 关注高IV排名的科技股
```

### 场景2：市场中性策略
```
1. 选择SPY、QQQ作为基准
2. 添加VIX进行波动率分析

```

### 场景3：行业轮动分析
```
1. 选择多个行业ETF（XLF、XLE、XLK等）
2. 对比不同行业的期权机会
3. 寻找隐含波动率排名差异
4. 制定行业轮动策略
```

## ⚠️ 注意事项

### 股票代码要求
- 必须是在美国交易所上市的股票
- 支持NYSE、NASDAQ等主要交易所
- 股票代码区分大小写（系统会自动转换为大写）
- 不支持场外交易(OTC)股票

### 期权可用性
- 并非所有股票都有活跃的期权市场
- 小盘股和新上市股票可能缺乏期权流动性
- 系统会自动过滤掉流动性不足的期权

### 性能优化
- 选择股票数量建议控制在20只以内
- 首次分析某只股票时可能需要较长时间（数据缓存）
- 后续分析会利用缓存，速度更快

## 📊 数据来源

- **期权数据**：实时期权链数据
- **市场数据**：VIX、SPY等指数数据
- **更新频率**：实时获取（有缓存机制）
