# 📈 股票代码输入指南

## 🎯 简洁易用的股票输入

美股期权卖方推荐工具采用简洁的设计，只需在侧边栏输入股票代码即可开始分析。

## 💡 使用方法

### 1️⃣ 输入股票代码
在侧边栏的"股票池"部分的文本框中输入股票代码，**每行一个**：

```
AAPL
MSFT  
TSLA
GOOGL
SPY
```

### 2️⃣ 验证并应用
点击"✅ 验证并应用"按钮：
- 🔍 系统自动验证每个股票代码的有效性
- ✅ 显示有效的股票代码数量
- ❌ 提示无效的股票代码（如果有）
- 🎯 将有效股票应用到分析列表

### 3️⃣ 开始分析
验证完成后，点击"🚀 开始分析"即可获得期权推荐。

## 📋 支持的股票类型

- **个股**：AAPL、MSFT、TSLA、GOOGL、AMZN等
- **ETF**：SPY、QQQ、IWM、XLF、GLD等  
- **指数**：DIA、VTI、EFA、EEM等
- **其他**：任何在美国交易所上市的标的

## 🔧 实用功能

### 当前分析股票
- 显示已验证的股票列表
- 显示股票数量和代码
- 绿色背景表示已准备好分析

### 清空功能
- 点击"🗑️ 清空股票列表"可重新开始
- 清空后可输入新的股票代码

## ⚠️ 注意事项

1. **股票代码格式**
   - 使用正确的美股代码（如AAPL而非苹果）
   - 大小写不敏感（自动转换为大写）
   - 每行只输入一个代码

2. **验证结果**
   - 绿色✅表示代码有效并已应用
   - 红色❌表示代码无效，请检查拼写
   - 只有有效代码会被用于分析

3. **分析建议**
   - 建议同时分析5-15只股票获得最佳体验
   - 过多股票可能导致分析时间较长
   - 可以多次运行，逐步调整股票列表
